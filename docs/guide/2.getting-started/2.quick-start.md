# Quick Start

## 1. Clone the git project
``` bash
git clone https://github.com/kinjal1993/VuePressDemo.git
``` 

## 2. Navigate to the folder
``` bash
cd <folder-name>
``` 

## 3. Install dependencies
``` bash
npm install
```

## 4. To add new markup files
``` bash
npm start
```

## 5. For development
``` bash
cd docs
vuepress dev
```

## 6. For production build
``` bash
cd docs
vuepress build
```

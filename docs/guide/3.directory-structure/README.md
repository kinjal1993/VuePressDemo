# Directory Structure for docs

The project follows the recommended structure by **VuePress** which is as follows:

```bash
├── docs
│   ├── .vuepress
│   │   ├── styles
|   │   │   ├── *.styl
|   |   ├── public
|   |   |   ├── assets
|   |   |   |   ├── logo.png
|   |   ├── config.js
|   |   ├── index.js
│   ├── guide
│   │   ├── **/*.md
|   |   ├── *.md
├── *.md
```
